# MiExamen

Este repositorio contiene un "examen" o conjunto de ejercicios básicos de JavaScript y HTML.

El examen se centra en los siguientes conceptos:
-   **Manipulación del DOM**: Interacción con elementos HTML usando JavaScript.
-   **`localStorage`**: Almacenamiento y recuperación de datos en el navegador utilizando `localStorage` y la serialización/deserialización de objetos JSON (`JSON.stringify`, `JSON.parse`).
-   **Lógica de programación**: Resolución de un problema algorítmico (contar círculos en los dígitos de un número).
-   **Estructura básica de HTML y CSS**: Organización de contenido y aplicación de estilos sencillos.
